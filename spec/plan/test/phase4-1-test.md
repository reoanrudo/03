# Phase 4-1: 難易度フィルタリングのテスト

## テスト目的
SongSelectorの難易度フィルタリング機能が正しく動作することを確認する。

## テスト手順

### 1. 開発サーバーを起動
```bash
cd frontend
npm run dev
```

### 2. ブラウザを開いてSongSelector画面を表示
```
http://localhost:3000
```

### 3. 「全て」フィルターを確認
- [ ] 「全て」ボタンが選択状態（灰色/オレンジ）
- [ ] 全ての曲（3曲）が表示されていること
  - きらきら星（初級）
  - Wonder Wall（中級）
  - Stairway to Heaven（上級）

### 4. 「初級」フィルターを確認
- [ ] 「初級」ボタンをクリック
- [ ] 初級曲のみが表示されていること
  - きらきら星（初級）のみ
- [ ] ボタンの色が緑（`bg-green-500`）であること
- [ ] 他の曲が非表示であること

### 5. 「中級」フィルターを確認
- [ ] 「中級」ボタンをクリック
- [ ] 中級曲のみが表示されていること
  - Wonder Wall（中級）のみ
- [ ] ボタンの色が黄色（`bg-yellow-500`）であること
- [ ] 他の曲が非表示であること

### 6. 「上級」フィルターを確認
- [ ] 「上級」ボタンをクリック
- [ ] 上級曲のみが表示されていること
  - Stairway to Heaven（上級）のみ
- [ ] ボタンの色が赤（`bg-red-500`）であること
- [ ] 他の曲が非表示であること

### 7. 「全て」に戻る
- [ ] 「全て」ボタンをクリック
- [ ] 全ての曲が再表示されていること

### 8. トランジション確認
- [ ] フィルター変更時のアニメーション（scale-110）がスムーズであること
- [ ] ボタンのホバー効果（`hover:scale-105`）が機能すること

## 期待される結果

- 各フィルターが正しく機能すること
- 色分けが正しく表示されること
- ユーザー体験がスムーズであること
